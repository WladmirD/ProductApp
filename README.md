# Ice Cream Ordering App

A Ice Cream ordering app made with javascript, node, express and mongodb.

## Getting Started

Clone the repository and extract it. 

### Prerequisites

MongoDB

* [Node.js]
* [Express]

* [Body Parser]
* [Json Web Token]
* [Mongoose]
* [BCrypt]
